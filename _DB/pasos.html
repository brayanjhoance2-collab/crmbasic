Guía de Configuración API

Obtener credenciales para conectar WhatsApp, Instagram, Facebook y Google Sheets a tu CRM profesional

WhatsApp Business API

  1. Crear cuenta de Meta Business

    Ve a business.facebook.com y crea tu cuenta de empresa.

  2. Solicitar WhatsApp Business API

    En Meta Business Manager:

    • Ve a "Cuentas" → "WhatsApp Business"
    • Solicita acceso a la API
    • Verifica tu empresa (puede tomar días)

  3. Configurar número de teléfono

    Una vez aprobado:

    • Agrega tu número de teléfono empresarial
    • Verifica el número con SMS/llamada
    • Copia el Phone Number ID

  4. Generar Access Token

    En la configuración de WhatsApp:

    • Ve a "Configuración" → "API Setup"
    • Genera un Access Token permanente
    • Guarda el token de forma segura

  5. Configurar Webhook

    Para recibir mensajes:

    • URL: https://tudominio.com/api/webhook/whatsapp
    • Token de verificación: crea uno personalizado
    • Suscríbete a "messages"

Credenciales necesarias:

Phone Number ID
ID del número verificado
Access Token
Token permanente de acceso
Webhook Verify Token
Token que tú creas
Business Account ID
ID de cuenta empresarial (opcional)
Importante: WhatsApp Business API es de pago después del período de prueba. Requiere verificación empresarial que puede tomar varios días.

Instagram Business

  1. Crear App en Facebook Developers

    Ve a developers.facebook.com:

    • Clic en "Crear App"
    • Selecciona "Empresa" como tipo
    • Completa información básica

  2. Agregar producto Instagram

    En tu app:

    • Ve a "Agregar producto"
    • Selecciona "Instagram Basic Display"
    • También agrega "Instagram Graph API"

  3. Configurar cuenta de Instagram

    Necesitas una cuenta empresarial:

    • Convierte tu Instagram personal a empresarial
    • Conecta con tu página de Facebook
    • Copia el Instagram Business Account ID

  4. Generar Access Token

    En Facebook for Developers:

    • Ve a "Herramientas" → "Explorador de Graph API"
    • Selecciona tu app y página
    • Genera token con permisos: instagram_basic, pages_read_engagement

  5. Configurar Webhook

    Para mensajes directos:

    • URL: https://tudominio.com/api/webhook/instagram
    • Token de verificación personalizado
    • Suscríbete a "messages"

Credenciales necesarias:

Instagram Business Account ID
ID de tu cuenta empresarial
Access Token
Token con permisos de Instagram
Webhook Verify Token
Token que tú creas
Ventaja: Instagram es más fácil de configurar que WhatsApp y es gratuito para uso básico.

Facebook Messenger

  1. Crear App en Facebook Developers

    Ve a developers.facebook.com:

    • Clic en "Crear App"
    • Selecciona "Empresa"
    • Guarda el App ID y App Secret

  2. Agregar Messenger

    En tu app:

    • Ve a "Agregar producto"
    • Selecciona "Messenger"
    • Configura Messenger

  3. Crear/Configurar Página

    Necesitas una página de Facebook:

    • Crea página empresarial en facebook.com/pages/create
    • Ve a Configuración → Información de la página
    • Copia el Page ID

  4. Generar Page Access Token

    En Messenger → Configuración:

    • Selecciona tu página
    • Genera Page Access Token
    • Guarda el token de forma segura

  5. Configurar Webhook

    En Messenger → Configuración → Webhooks:

    • URL: https://tudominio.com/api/webhook/facebook
    • Token de verificación personalizado
    • Suscríbete a: messages, messaging_postbacks

  6. Solicitar permisos

    En Revisión de aplicaciones:

    • pages_messaging
    • pages_show_list
    • pages_read_engagement

Credenciales necesarias:

Page ID
ID de tu página de Facebook
Page Access Token
Token de acceso de la página
App ID
ID de tu aplicación
App Secret
Clave secreta de la app
Webhook Verify Token
Token que tú creas
Importante: Facebook requiere que tu app esté en modo desarrollo inicialmente. Para producción necesitas aprobación de Facebook.

Google Sheets API

  1. Crear proyecto en Google Cloud Console

    Ve a console.cloud.google.com:

    • Clic en "Crear proyecto" o selecciona uno existente
    • Nombra tu proyecto (ej: "CRM-Sheets-Integration")
    • Selecciona organización si aplica
    • Clic en "Crear"

  2. Habilitar APIs necesarias

    En el panel de Google Cloud Console:

    • Ve a "APIs y servicios" → "Biblioteca"
    • Busca y habilita: Google Sheets API
    • Busca y habilita: Google Drive API
    • Opcionalmente: Google Apps Script API

  3. Configurar pantalla de consentimiento OAuth

    Antes de crear credenciales:

    • Ve a "APIs y servicios" → "Pantalla de consentimiento de OAuth"
    • Selecciona "Externo" (para cualquier usuario Gmail)
    • Completa información básica: nombre de app, email de soporte
    • Agrega dominio autorizado: tudominio.com
    • Scopes requeridos se agregarán automáticamente

  4. Crear credenciales OAuth 2.0

    En "APIs y servicios" → "Credenciales":

    • Clic en "Crear credenciales" → "ID de cliente de OAuth 2.0"
    • Tipo de aplicación: Aplicación web
    • Nombre: "CRM Google Sheets Client"
    • Orígenes autorizados: https://tudominio.com
    • URI de redirección: https://tudominio.com/api/auth/google/callback

  5. Obtener credenciales

    Después de crear el cliente OAuth:

    • Descarga el archivo JSON o copia las credenciales
    • Anota el Client ID
    • Anota el Client Secret
    • Confirma la Redirect URI

  6. Configurar en tu CRM

    En la sección Google Sheets de tu CRM:

    • Ve a "Configurar API" (solo admins)
    • Pega el Client ID
    • Pega el Client Secret
    • Confirma la Redirect URI
    • Guarda la configuración

  7. Probar la conexión

    Para verificar que todo funciona:

    • Clic en "Conectar con Google"
    • Autoriza los permisos solicitados
    • Verifica que aparezca "Conectado"
    • Prueba listar tus hojas de cálculo

Credenciales necesarias:

Client ID
Identificador público del cliente OAuth (ej: 123456789-abc.apps.googleusercontent.com)
Client Secret
Clave secreta del cliente OAuth (ej: GOCSPX-xxxxxxxxxxxxxxxxxxxxxxxx)
Redirect URI
URL de callback autorizada (debe coincidir exactamente)
Ventajas 2025: Google Sheets API es gratuito hasta 100 requests por segundo. Perfecto para sincronizar contactos, generar reportes automáticos y backup de datos.
Nuevos requisitos 2025: Google requiere verificación de dominio para aplicaciones en producción. Asegúrate de tener SSL válido y dominio verificado.

Requisitos Generales

Dominio con SSL: Todas las plataformas requieren HTTPS (certificado SSL válido)

Verificación empresarial: Puede ser requerida para funcionalidades completas

Tiempo de aprobación: WhatsApp puede tomar varios días, Instagram y Facebook usualmente son inmediatos

Google Sheets: Configuración inmediata, pero requiere verificación de dominio para producción

Backup de datos: Google Sheets es ideal para crear respaldos automáticos de tu CRM
