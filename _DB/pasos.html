Guía Actualizada APIs 2025

Configuración paso a paso para WhatsApp Business API, Instagram Business, Facebook Messenger y Google Sheets API con información actualizada de septiembre 2025

WhatsApp Business API

  1. Crear Meta Business Portfolio

    Ve a business.facebook.com y crea tu Portfolio Comercial.

    • Haz clic en "Crear Portfolio Comercial"
    • Completa información empresarial
    • Verifica tu identidad con documentos oficiales

  2. Solicitar WhatsApp Business API

    En tu Portfolio Comercial:

    • Ve a "Configuración" → "Activos Comerciales"
    • Haz clic en "Agregar" → "Cuenta de WhatsApp Business"
    • Selecciona "Crear nueva cuenta"
    • Completa verificación empresarial (3-7 días)

  3. Configurar Número de Teléfono

    Una vez aprobada tu cuenta:

    • Ve a "WhatsApp Manager"
    • Selecciona "Números de Teléfono" → "Agregar"
    • Verifica tu número con SMS/llamada
    • Copia el ID del Número de Teléfono

  4. Generar Token de Acceso

    En WhatsApp Manager:

    • Ve a "Configuración de API" → "Tokens"
    • Genera token permanente
    • Configura permisos: whatsapp_business_messaging
    • Guarda el Token de Acceso

  5. Configurar Webhook

    Para recibir mensajes:

    • URL: https://tudominio.com/webhook/whatsapp
    • Token de verificación: cadena única
    • Eventos: messages, message_deliveries

Credenciales Necesarias:

ID del Número de Teléfono
Identificador único de tu línea WhatsApp Business
Token de Acceso de WhatsApp Business
Token permanente para llamadas API
Token de Verificación Webhook
Cadena secreta para verificar webhooks
ID de Cuenta WhatsApp Business
Identificador de cuenta empresarial
Costos 2025: WhatsApp cobra por conversación. Primeras 1000 conversaciones de servicio gratis/mes. Marketing desde €0.071 por conversación.

Instagram Business API

  1. Crear Aplicación Facebook

    Ve a developers.facebook.com:

    • Haz clic en "Mis Apps" → "Crear App"
    • Selecciona "Empresa"
    • Guarda ID de App y Clave Secreta

  2. Agregar Instagram Basic Display

    En tu aplicación:

    • Ve a "Productos" → "Instagram Basic Display"
    • También agrega "Instagram Graph API"
    • Configura URIs de redirección OAuth

  3. Conectar Instagram con Facebook

    Requisito obligatorio:

    • Instagram debe ser cuenta "Professional"
    • Conecta con Página Facebook empresarial
    • Ve a Instagram → Configuración → Cuenta Profesional
    • Copia ID de Cuenta Comercial Instagram

  4. Generar Token de Acceso

    En Graph API Explorer:

    • Selecciona tu app y página
    • Permisos: instagram_basic, pages_read_engagement
    • Genera Token de Acceso de Página

  5. Configurar Webhook (Nuevo 2025)

    Para mensajes directos:

    • URL: https://tudominio.com/webhook/instagram
    • Eventos: messages, messaging_postbacks

Credenciales Necesarias:

ID de Cuenta Comercial Instagram
Número único de cuenta Business
Token de Acceso de Página
Token con permisos Instagram
ID de App Facebook
Identificador de aplicación
Clave Secreta de App
Clave privada aplicación
Nuevo 2025: Instagram API ahora soporta programación de Reels y Stories. Mensajería DM incluida para cuentas Business verificadas.

Facebook Messenger API

  1. Crear App Facebook

    En developers.facebook.com:

    • Crear nueva aplicación "Empresa"
    • Nombre: "Mi CRM Messenger Bot"
    • Guardar ID de App y Clave Secreta

  2. Agregar Messenger Platform

    En tu aplicación:

    • Ve a "Productos" → "Messenger"
    • Haz clic en "Configurar"
    • Configura la plataforma Messenger

  3. Crear/Configurar Página Facebook

    Necesitas página empresarial:

    • Crea página en facebook.com/pages/create
    • Ve a Configuración → Información General
    • Copia el ID de Página

  4. Generar Token de Acceso de Página

    En configuración Messenger:

    • Sección "Tokens de Acceso"
    • Selecciona tu página
    • Genera Token de Acceso de Página
    • Configurar permisos: pages_messaging

  5. Configurar Webhook

    En configuración Messenger:

    • URL: https://tudominio.com/webhook/facebook
    • Token verificación personalizado
    • Eventos: messages, messaging_postbacks
    • Suscribir página al webhook

  6. Solicitar Permisos (Producción)

    Para uso público:

    • Ve a "Revisión de App"
    • Solicitar: pages_messaging
    • Solicitar: pages_read_engagement
    • Completar proceso de revisión

Credenciales Necesarias:

ID de Página
Identificador de página Facebook
Token de Acceso de Página
Token para enviar mensajes
ID de App
Identificador aplicación
Clave Secreta de App
Clave privada aplicación
Token Verificación Webhook
Token personalizado webhook
Limitación 2025: Messenger tiene ventana 24h para responder. Después solo con etiquetas específicas: HUMAN_AGENT, POST_PURCHASE_UPDATE, ACCOUNT_UPDATE.

Google Sheets API

  1. Crear Proyecto Google Cloud

    Ve a console.cloud.google.com:

    • Haz clic en "Nuevo Proyecto"
    • Nombre: "CRM-Sheets-Integration"
    • Selecciona organización si aplica

  2. Habilitar APIs Necesarias

    En Google Cloud Console:

    • Ve a "APIs y Servicios" → "Biblioteca"
    • Buscar y habilitar: Google Sheets API
    • Buscar y habilitar: Google Drive API

  3. Configurar Pantalla Consentimiento OAuth

    Antes de crear credenciales:

    • Ve a "APIs y Servicios" → "Pantalla de Consentimiento OAuth"
    • Selecciona "Usuario Externo"
    • Completa información app: nombre, email soporte
    • Agregar dominio autorizado: tudominio.com

  4. Crear Credenciales OAuth 2.0

    En "APIs y Servicios" → "Credenciales":

    • Haz clic en "Crear Credenciales" → "ID de cliente OAuth 2.0"
    • Tipo: Aplicación Web
    • Orígenes autorizados: https://tudominio.com
    • URI redirección: https://tudominio.com/auth/google/callback

  5. Obtener Credenciales

    Después de crear cliente OAuth:

    • Descargar archivo JSON o copiar credenciales
    • Anotar ID de Cliente
    • Anotar Secreto de Cliente
    • Confirmar URI de Redirección

  6. Configurar en tu Sistema

    En configuración de tu aplicación:

    • Agregar ID de Cliente OAuth
    • Agregar Secreto de Cliente
    • Configurar URI de redirección exacta
    • Probar flujo de autorización

  7. Probar Conexión

    Verificar funcionamiento:

    • Iniciar flujo OAuth
    • Autorizar permisos solicitados
    • Verificar acceso a Sheets
    • Probar lectura/escritura datos

Credenciales Necesarias:

ID de Cliente
Identificador público OAuth (ej: 123456789-abc.apps.googleusercontent.com)
Secreto de Cliente
Clave secreta OAuth (ej: GOCSPX-xxxxxxxxxxxxxxxxxxxxxxxx)
URI de Redirección
URL callback autorizada (debe coincidir exactamente)
Ámbitos (Scopes)
https://www.googleapis.com/auth/spreadsheets, https://www.googleapis.com/auth/drive
Ventajas 2025: Google Sheets API gratuito hasta 100 requests/segundo. URLs de autorización actualizadas: oauth2.googleapis.com/token
Nuevos Requisitos 2025: Google requiere verificación dominio para producción. URLs OAuth actualizadas. Política privacidad obligatoria.

Requisitos Generales 2025

Dominio con SSL: Todas las APIs requieren HTTPS válido y certificado SSL

Verificación Empresarial: Meta requiere documentos oficiales para WhatsApp/Instagram/Facebook

Tiempo de Aprobación: WhatsApp 3-7 días, Instagram/Facebook inmediato, Google inmediato

Nuevas Políticas 2025: Controles anti-spam más estrictos, verificación dominio obligatoria

Costos: WhatsApp por conversación, Instagram/Facebook gratuito uso básico, Google Sheets gratuito

Webhooks: Todos requieren HTTPS, tokens de verificación únicos y respuestas rápidas
